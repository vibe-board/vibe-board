## What is Vibe Kanban?

Vibe Kanban is an **orchestration platform for AI coding agents** that helps developers plan, review, and safely execute AI-assisted coding tasks. Each task runs in an isolated git worktree, giving you complete control over your codebase whilst leveraging the power of AI assistants.

## About This Fork

This repository is a **fork of [vibe-kanban](https://github.com/BloopAI/vibe-kanban)**. The upstream project introduced a new UI that we found too complex for our workflow. We prefer the **older UI** for its simplicity and ease of use. This fork maintains a version of Vibe Kanban based on the **legacy UI**, so you can continue using the interface you're familiar with.

## Why Vibe Kanban?

  <Card title="Safe AI Execution" icon="shield-check">

## New Features in This Fork

This fork adds the following features on top of the legacy UI:

- **Generate merge commit message by agent** — Use an AI agent to draft merge commit messages for you.
- **Commit after linter changed** — Automatically commit when the linter has made changes to your code.
