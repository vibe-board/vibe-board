## Managing Agents

### Enabling and Disabling Agents

You can control which agents appear in the agent selector:

1. Go to **Settings → Agents**
2. Toggle the checkbox next to each agent to enable or disable it
3. Only enabled agents will appear when creating task attempts

### Agent Ordering

Customise the order in which agents appear in selectors:

1. Go to **Settings → Agents**
2. Use the drag handle to reorder agents
3. Your preferred agents can be moved to the top for quick access

<Note>
Agent ordering and enable/disable settings are stored in your configuration and persist across sessions.
</Note>

## Agent Capabilities

Different agents support different capabilities:

| Capability | Description | Supported Agents |
|-----------|-------------|-----------------|
| **Session Fork** | Continue from an existing conversation | All ACP agents |
| **AskUserQuestion** | Ask structured questions during execution | Claude Code, ACP agents |
| **Message Editing** | Edit previous messages to redirect the agent | Claude Code, Amp, Codex, Gemini, Qwen |
| **Action Approvals** | Require human approval for certain actions | Codex |
